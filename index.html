About

I am working on this website for my assessment by creating an environmental advisory page promoting Kaitiakitanga or Guardianship. I will use this website to promote conservation efforts and educate people about New Zealand's various Jewelled geckos species.

Why am I doing this?

I am doing this to show that not only I can create a website using the things I learnt in class about HTML and CSS. But I can also create a website that promotes Kaitiakitanga and allowing others to be educated about endangered the Jewelled gecko species of New Zealand.

My Mission

My mission is to educate everyone in Mount Roskill Grammar School about the many endangered Jewelled gecko species of New Zealand. It is also my mission to make this website for an assessment to show that I can create a website using what I learnt in class.
